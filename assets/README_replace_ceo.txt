Para usar la foto que subiste en el chat como imagen del CEO: coloca el archivo de imagen en esta carpeta y nómbralo `ceo.jpg` (ruta relativa: `assets/ceo.jpg`).

Si prefieres otro nombre, actualiza el `src` del elemento con id `ceoImg` en `index.html`.

Nota: hay un placeholder SVG (`assets/ceo.svg`) que se mostrará hasta que coloques `assets/ceo.jpg`.
